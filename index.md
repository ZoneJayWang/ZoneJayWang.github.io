# [](#header-1)十月十一日
今天我准备离开公司。谢谢所有人。再见是为了更好的相遇。

我现在坐在办公桌前，等待下班的时间。

无疑我可以早走，但我不想这样。

看着屏幕，没有任何想要看的东西。

无论声色，无论有无，也不会汲汲营营。

但我行动仍然很快，我心中很慢。

在外，在内。

有，无。

我，你。

现在只差你，才能无我。

那叫爱。

2017/10/11 16：25 于陶然亭
